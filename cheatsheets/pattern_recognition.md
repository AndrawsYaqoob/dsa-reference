# Pattern Recognition

Given a signal in the problem, use this to identify which pattern to reach for.

---

## By Problem Signal

| Signal in Problem                                      | Pattern                  |
|--------------------------------------------------------|--------------------------|
| Sorted array, find target                              | Binary Search            |
| Sorted array, find pair that satisfies condition       | Two Pointers             |
| Find pair that sums to target                          | Two Pointers / Hashmap   |
| Contiguous subarray or substring                       | Sliding Window           |
| Subarray of fixed size K                               | Sliding Window (fixed)   |
| Longest / shortest subarray with constraint            | Sliding Window (variable)|
| Duplicates, frequencies, lookup in O(1)                | Hashmap                  |
| All possible subsets                                   | Backtracking             |
| All possible permutations                              | Backtracking             |
| All combinations of K elements                         | Backtracking             |
| "All possible", "generate all", "find all"             | Backtracking             |
| Shortest path, unweighted graph                        | BFS                      |
| Shortest path, weighted graph                          | Dijkstra (BFS + Heap)    |
| Explore all paths, detect cycle                        | DFS                      |
| Connected components                                   | DFS / Union Find         |
| Tree traversal, path problems                          | DFS (Recursion)          |
| Level by level traversal                               | BFS                      |
| "Min / max", "how many ways", "is it possible"         | Dynamic Programming      |
| Overlapping subproblems                                | Dynamic Programming      |
| Comparing two sequences                                | 2D DP                    |
| Selecting items under a constraint                     | Knapsack DP              |
| Repeated min / max from a changing collection          | Heap                     |
| Top K elements                                         | Heap                     |
| Nested structures, undo / redo, backtracking           | Stack                    |
| Next greater / smaller element                         | Monotonic Stack          |
| Task scheduling, process in order                      | Queue                    |
| Prefix matching, autocomplete                          | Trie                     |
| Cycle detection in linked list                         | Fast & Slow Pointers     |
| Find middle of linked list                             | Fast & Slow Pointers     |
| Search rotated sorted array                            | Binary Search (rotated)  |
| Find min / max value that satisfies a condition        | Binary Search on Answer  |
